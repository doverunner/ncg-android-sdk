상세한 설명

HLS 컨텐츠 m3u8파일내 key에 대한 정보를 추출할 수 있는 기능의 클래스이다.

HLS 컨텐츠는 m3u8파일을 통해 재생되지만 실제로 암호화되어있는 파일은 m3u8파일내 기록된 key URL 경로에 해당되는 키 파일이다. 따라서 m3u8파일만으로는 DRM 적용여부 및 DRM 정보를 확인할 수 없다. 해당 클래스는 m3u8파일을 입력받아 DRM 적용여부 및 DRM정보를 제공하는 기능을 제공한다.
