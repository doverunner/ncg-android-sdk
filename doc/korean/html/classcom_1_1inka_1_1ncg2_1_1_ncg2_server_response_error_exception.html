상세한 설명

License Issuer 서버에서 명시적으로 오류를 Response한 경우 해당 예외가 발생된다.
NCG 서버에서 Client의 요청을 처리하다가 오류를 반환한 경우 해당 예외가 발생되게 된다.
예륻들어, 라이선스 요청시 기기횟수가 초과되었다거나 요청에 대한 Signature가 맞지 않은 등의 명확한 오류코드를 서버로부터 전달받게되면
해당 예외를 통해 오류를 확인할 수 있다.
