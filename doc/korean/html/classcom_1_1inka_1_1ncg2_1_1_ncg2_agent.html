상세한 설명

이 클래스는 NCG 라이브러리의 초기화 및 컨텐츠의 라이선스처리등의 NCG SDK의 핵심기능을 지원한다. NCG SDK에서 내부적으로 이 인터페이스를 Single Instance 형태로 구현한다. 따라서 Ncg2Agent 인터페이스는 SDK외부에서 구현하는 인터페이스가 아니라 SDK내부에서 구현된 객체에 접근할때 사용되어지는 인터페이스이다. 해당 Single Instance의 접근은 별도의 Static 메소드에 의해 제공되는데 Ncg2Agent 인터페이스의 객체는 Ncg2Player,Ncg2PlayerFactory클래스의 getNcgAgentInstance()메소드에 의해 획득되어질 수 있다.

해당 인터페이스에서 제공하는 주요기능은 라이선스처리와 NCG컨텐츠 복호화이다. 라이선스내에는 NCG 컨텐츠의 재생권한정보가 포함되어있으며 NCG 컨텐츠를 복호화할 수 있는 CEK(Cotent Encryption Key)가 포함되어있으므로 재생시키기 위해서는 라이선스가 반드시 있어야 한다. 서버로부터 라이선스를 받게되면 별도의 라이선스 파일에 저장되어 Application이 재실행되어도 라이선스는 유지되며 Application이 재설치된다면 라이선스는 다시 다운로드되어야 된다. 라이선스파일은 Application이 설치된 경로에 생성된다.(예:/data/data/(package name)/files/)

NCG 컨텐츠의 복호화는 NcgFile 내부 인터페이스에 의해 제공되며 createNcgfile메소드에 의해 해당 클래스의 인스턴스를 생성할 수 있다. 자체적으로 Player를 제작하지 않는다면 해당 NcgFile 인터페이스를 사용할 필요는 없고 재생을 위해서는 Ncg2Player 혹은 Ncg2MediaPlayer 를 이용할 수 있다.

서버와 통신하여 시리얼 넘버를 인증한다.

매개변수
contextcontext 객체
deviceModel허용할 단말 모델명. 모델명은 [설정 - 폰/태블릿 정보 - 모델 번호] 또는 유사한 메뉴에서 확인 가능하며 대소문자를 구분하여 정확하게 기입해야 한다. 해당 모델명에 추가되지 않은 단말을 사용할 경우 exception이 발생한다.
