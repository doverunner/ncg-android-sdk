상세한 설명

JNI 로컬웹서버에서 발생될 수 있는 이벤트를 Listener 형태로 전달받기 위한 클래스 실제로 SDK외부에 노출되는 API는 아니며 SDK를 이용하는 Client는 Ncg2LocalWebServer.WebServerListener 인터페이스를 이용해야 한다.

NOTE: JNI에서 받는 메소드를 static 콜백을 만든 이유 JNI에서 받는 메소드를 static 콜백을 만든 이유는 특정 VM이 뻣는 증상이 발생되기 때문이다. JNI에서 보관하는 객체의 Reference가 어느 순간 Invalid되는 문제가 있다. 이 문제를 해결하기 위해 객체를 선언할때 volatile, static 키워드를 이용해보기도 했지만 소용없었다.
