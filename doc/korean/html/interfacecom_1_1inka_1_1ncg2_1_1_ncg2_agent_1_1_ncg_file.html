상세한 설명

이 인터페이스는 NCG 파일을 복호화하는 인터페이스를 제공한다. Ncg2Agent.createNcgFile 메소드에 의해 생성된다. NOTE: 이 인터페이스의 객체를 여러개 생성할 수 있으므로, 멀티 Open 처리가 가능해야 하지만 JNI Core에서 다중을 Open 지원하지 않으므로 NcgFile 객체를 여러개 생성해서 사용하면 안된다. 다중 Open 기능은 추후 지원될 예정이다.

파일을 읽을 파일포인터의 위치를 이동시킨다.

매개변수
offset파일포인터를 이동시킬 오프셋
seekMethod기준점. SeekMethod의 값 중 하나를 지정해야 한다. Begin: 파일 시작 지점. fseek()의 SEEK_SET과 동일하다. Current: 현재 파일포인터의 위치. fseek()의 SEEK_CUR과 동일하다. End: 파일 끝 지점. fseek()의 SEEK_END와 동일하다.
예외
