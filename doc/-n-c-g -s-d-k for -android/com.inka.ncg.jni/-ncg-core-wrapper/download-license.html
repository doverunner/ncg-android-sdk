public int downloadLicense(String userID, String cid, String siteID, String oid, String appID, String acquisitionURL, boolean temporary, HttpRequestJniCallback callback)

so 내부적으로 HTTP 통신을 처리하여 License를 다운로드 받는다. 통신시 암화방식은 내부적으로 PKI 방식이 이용된다. NOTE: 특이하게도 callback 객체를 미리등록시켜놨다가 사용하려고 시도를 했었는데 callback 객체를 미리 등록시켜놨다가 사용하려고 하면 JNI에서 오류가 발생되었다. 따라서 DownloadLicense메소드에서 직접 파라미터를 받도록 처리하였다.

Return

오류코드. NCGERR_FAILED_TO_DOWNLOAD_CERTIFICATE : 인증서를 다운로드하는 오류가 발생됨을 나타낸다. NCGERR_FAILED_TO_DOWNLOAD_LICENSE : 라이선스RO를 다운로드하는 오류가 발생됨을 나타낸다.NCGERR_LICENSE_SERVER_RESPONSE_ERROR : 서버로부터 명시적인 오류코드를 받은 경우를 나타낸다.
