Ncg2ServerResponseErrorException

public class Ncg2ServerResponseErrorException extends Ncg2Exception

The relevant exception occurs when the error responses on License Issuer explicitly And it also occurs when the error is returned during processing the client requests on NCG server. For example, if the corrected error code is received from the server, it is able to check error through the relevant exception such as device number is overloaded when it requests license and it is false Signature about requesting.
