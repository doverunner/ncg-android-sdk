LocalFileCallback

public interface LocalFileCallback

This interface provides the necessary interface to implement local file requests as a feature outside of the SDK. For handling local file requests on app, receive the relevant interface and implement it. And it is able to register callback through Ncg2Agent.setLocalFileCallback method.
