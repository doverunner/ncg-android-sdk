sendCustomChannelRequest

public abstract String sendCustomChannelRequest(String acquisitionUrl, String requestData)

It sends a custom request to a license server. By the custom request, a secure channel based on PKI is established. As a result, NCG library can communicate with the license server on turst. It sends the custom request to the license server, and then the license server redirect the request to gateway-page to process the request. The response data that gateway-page responded is returned to the license server, and then license server relays the data to NCG library.

Return

Response string received from the license server

Parameters

acquisitionUrl

License server URL

requestData

Request data

Throws

Ncg2HttpException

If an error is occurred while communicating with license server.

Ncg2Exception

If an error is occurred while sending custom data.
