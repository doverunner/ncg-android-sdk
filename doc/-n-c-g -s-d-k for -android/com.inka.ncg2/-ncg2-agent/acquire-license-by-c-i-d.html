public abstract void acquireLicenseByCID(String contentID, String userID, String orderID, String acquisitionURL)

It acquires a license correspond to Content ID to license sever. It can be used when the app has no NCG content but knows Content ID and additional information for license acquisition. For example, it can be used to acquire a license of HLS contents.

Parameters

contentID

Content ID

userID

User ID

orderID

Order ID (optional). It is necessary information to order and it is used only for a special cases.

acquisitionURL

License acquisition URL

Throws

Ncg2ServerResponseErrorException

If the license server returns an error code.

Ncg2HttpException

If an error is occurred while communicating with the license server.

Ncg2DeviceIdException

If a device reboot and restart of the media server is required

Ncg2Exception

If an error is occurred while acquiring license.
