serialAuthentication

public abstract int serialAuthentication(Context context, Array<String> deviceModel, File file, String idValue)

It authenticate a serial number using server

Return

0 : Authentication success. The app may acquire license and do next step. 1 : Non registered serial number. The user must input correct serial number. 2 : Invalid Content ID or serial. The app must check NCG content file. 3 : Device registration exceeded. No more device can be registered. 4 : Authentication unavailable. Please contact INKA Entworks.

Parameters

context

Context object

deviceModel

Device model name to allow. The name can be found in [Settings - About phone/tablet - Model Number] or similar menu and it is case sensitive. An exception will be thrown if the app uses a name not included in 'deviceModel'.

file

NCG content file to authenticate

idValue

Serial number given by user

Throws

Ncg2DeviceNotMatchedException

If device model is not allowed

Ncg2HttpException

If an error is occurred while communicating server.

Ncg2Exception

If an error is occurred while en/decrypting, parsing data, or etc.
