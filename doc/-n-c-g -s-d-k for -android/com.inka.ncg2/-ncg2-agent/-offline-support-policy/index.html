Not support offline mode. If this value set and device is off-line status then OnInitializeCallbackListener.onError() method is called with OfflineNotSupported

OfflineSupport

Support off-line playback. When this value is set it is possible to play a content in off-line. Assume that the content is expired, If the user rolls back the device time then OnInitializeCallbackListener.onError() method is called.
