Ncg2LocalWebServer

public interface Ncg2LocalWebServer

LocalWebServer class for play. By using this class, you can acquire URL to set and play NCG files in local web server. Play URL can be played with a general player. Instance of the applicable interface can be obtained by getLocalWebserver() method of Ncg2Agent.
