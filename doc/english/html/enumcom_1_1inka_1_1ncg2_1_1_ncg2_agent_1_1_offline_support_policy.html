Detailed Description

LicenseValidation type shows a policy of off-line support. this enum class is used at Ncg2Agent.init() method to determine how to support off-line playback.

◆ OfflineSupport

com.inka.ncg2.Ncg2Agent.OfflineSupportPolicy.OfflineSupport =(2)

Support off-line playback. When this value is set it is possible to play a content in off-line. Assume that the content is expired, If the user rolls back the device time then OnInitializeCallbackListener.onError() method is called.
