Detailed Description

This interface provides interfaces to decrypt a NCG file. It is provided by Ncg2Agent.createNcgFile method. NOTE: Many objects of this interface can be created, so it can handle multi-open. However, JNI core cannot support multi-open functionality. An app must create only one object, and Multi-open feature will be supported soon.
