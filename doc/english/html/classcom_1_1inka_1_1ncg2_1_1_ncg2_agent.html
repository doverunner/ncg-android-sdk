Detailed Description

This class supports main function of NCG SDK such as contents license processing and initialization of NCG library. This interface is implemented as Single Instance form in NCG SDK internally. So Ncg2Agent is the interface to be used to approach to object implemented inside of SDK, not outside of SDK. The approaching of relevant Single Instance is provided by separated static method. And Ncg2Agent interface object can be acquired by getNcgAgentInstance() method of Ncg2Player,Ncg2PlayerFactory class. The main function for the relevant interface is to process licenses and decrypt NCG contents. License includes the play rights information of NCG contents and CEK(Content Encryption Key) for decrypting NCG contents. That is why users must have the license to play it. When it is received a license from the sever, the separate license file is saved. And even if the application is re-started, the license is maintained. But if application is re-installed, the license has to be re-downloaded. License file is created on the path where the application is installed. (e.g./data/data/(package name)/files/)

Decryption of NCG contents is provided by NcgFile inner interface. And it can be created the instance of relevant class by createNcgfile method. If it does not produce a Player by itself, it does need to use relevant NcgFile interface. And it is available to use Ncg2Player or Ncg2MediaPlayer to play it.

◆ getCurrentSecureTime()

abstract String com.inka.ncg2.Ncg2Agent.getCurrentSecureTime ( )
abstract

It gets UTC formatted current time.

Secure Time is used to compare license validity and current time. And it is updated in every second internally. An app may call this method to check current time in trust. It is based on UTC +- 0 (GMT), So it shows different value with local time. For example, KST (Korea Standard Time) uses UTC + 09:00 time zone. The app can get KST by adding 9 hours to return value.

Returns
UTC formatted current time (2012-12-31T00:00:00Z, "yyyy-MM-ddTHH:mm:ssZ")
