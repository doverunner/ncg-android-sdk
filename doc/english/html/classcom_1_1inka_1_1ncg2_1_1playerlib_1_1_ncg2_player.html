Detailed Description

This provides the common interface of MediaPlayer calss that is provided from software player and android.
Using the common interface,
there is no needs to be divide whether it is Mediaplayer of Android or software player in actual.
It is made for interface of this class to be similar with MediaPlayer class of android.
So if you are used to use MediaPlayer class, there is no difficulty to use InkaNcg2Player class.

To create an object of this class, it has to be used Ncg2PlayerFactory class.
